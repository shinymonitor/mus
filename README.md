# My Useful Stuff

Extracted from [construct.h](https://github.com/shinymonitor/construct.h)

- `log.h`: Logging to stdout and file
- `sv.h`: String slices/views
- `da.h`: Dynamic arrays, string builder and temp string functions
- `fh.h`: File handling (depends on `da.h` and `log.h`)
- `misc.h`: Utils, short circuiting and timing
